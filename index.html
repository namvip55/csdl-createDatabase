Bước 2 : Tạo các table


+ Sử dụng câu lệnh : create table name_table

+ Xác định typedef và primary key của các thuộc tính bên trong table

- Mã của thuộc tính nên để là int identity (khoi_tao,buoc_nhay)

- Số điện thoại or CCCD nên để kiểu char (n) ; String còn lại để varchar (n) , Tiếng việt có dấu để nchar (n) or nvarchar (n)

- Creat khóa chính : constraint PK_nameTable primary key (name_thuocTinh)

- Nếu table trên nó có thuộc tính giống nhau , thì thuộc tính table dưới phải kế thừa từ table trên !! Sử dụng cú pháp : name_thuocTinh typedef references name_table(name_thuocTinh) or constraint FK_nameTable2 foreign key (name_thuocTinh) references nameTable1 (name_thuocTinh)
